## Privacy Policy

**App name:** Razl — Plan Fast  
**Last updated:** May 2026

### Overview

Razl is a personal productivity dashboard. **Your data never leaves your device.**

---

### Data Storage

All data you create in Razl — tasks, habits, skills, stats, profile, movies, games, settings — is stored exclusively on your device using **IndexedDB**, a standard browser technology.

- No data is collected by the developer
- No data is transmitted to any server
- No analytics or tracking tools are used
- No account required
- No internet connection required (offline mode)

---

### What Data is Stored Locally

| Data | Purpose |
|---|---|
| Tasks and task history | Productivity tracking |
| Recurring task settings | Daily task automation |
| Habits and completion records | Habit tracking |
| Skills and progress | Skill progression |
| Stats (Health, Mood, Stamina, Motivation) | Character stats |
| Profile (name, avatar photo) | Hero profile |
| Pomodoro settings and timer state | Focus timer |
| Movies and watch status | Personal watchlist |
| Games and completion status | Personal game library |
| Theme and language preferences | UI settings |
| Daily XP and activity logs | Progress history |

---

### Notifications

If you grant notification permission, Razl sends **local notifications only**:

- Alert when a Pomodoro session or break ends
- Daily reminder at 20:00 about unfinished tasks and habits

No notification data is sent to external servers. Notifications are generated entirely on your device.

---

### Backup Files

The **Export** feature creates a file named `razl-backup.json` and saves it to your device. This file is for your personal use only. The developer has no access to it.

---

### Avatar and Media

Photos uploaded as your avatar are stored locally in IndexedDB. They are not uploaded to any server.

---

### Third-Party Services

Razl does not integrate with any third-party analytics, advertising, or data collection services.

The web version is hosted on **GitHub Pages**. GitHub may collect standard server logs (IP, browser type) as part of their hosting. See [GitHub's Privacy Policy](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement).

---

### Data Deletion

- **Web browser**: clear site data for nstp3.github.io in browser settings
- **Android APK**: uninstall the app, or go to Settings → Apps → Razl → Storage → Clear Data

---

### Contact

Open-source personal project. Questions? Open an issue on [GitHub](https://github.com/Nstp3/nstp3.github.io).

## Политика конфиденциальности

**Название приложения:** Razl — Plan Fast  
**Последнее обновление:** Май 2026

### Обзор

Razl — личный дашборд продуктивности. **Ваши данные никогда не покидают ваше устройство.**

---

### Хранение данных

Все данные в Razl — задачи, привычки, навыки, статы, профиль, фильмы, игры, настройки — хранятся исключительно на вашем устройстве с помощью **IndexedDB**.

- Разработчик не собирает никаких данных
- Данные не передаются ни на какие серверы
- Не используются аналитика или инструменты отслеживания
- Регистрация не требуется
- Интернет не нужен для работы (офлайн-режим)

---

### Какие данные хранятся локально

| Данные | Назначение |
|---|---|
| Задачи и история задач | Отслеживание продуктивности |
| Настройки повторяющихся задач | Ежедневная автоматизация |
| Привычки и записи выполнения | Трекер привычек |
| Навыки и прогресс | Прокачка умений |
| Статы (Здоровье, Настроение, Выносливость, Мотивация) | Характеристики персонажа |
| Профиль (имя, фото аватара) | Профиль героя |
| Настройки помодоро и состояние таймера | Таймер фокуса |
| Фильмы и статус просмотра | Личный список фильмов |
| Игры и статус прохождения | Личная библиотека игр |
| Тема и язык | Настройки интерфейса |
| Дневной XP и логи активности | История прогресса |

---

### Уведомления

Если вы предоставите разрешение, Razl отправляет **только локальные уведомления**:

- Оповещение об окончании сессии или перерыва помодоро
- Ежедневное напоминание в 20:00 о задачах и привычках

Никакие данные об уведомлениях не передаются на внешние серверы.

---

### Файлы бэкапа

Функция **Экспорт** создаёт файл `razl-backup.json` на вашем устройстве. Только для вашего личного использования. Разработчик не имеет доступа к этому файлу.

---

### Аватар и медиафайлы

Фотографии, загруженные как аватар, хранятся локально в IndexedDB. Не загружаются на серверы.

---

### Сторонние сервисы

Razl не интегрируется ни с какими сторонними сервисами сбора данных или аналитики.

Веб-версия размещена на **GitHub Pages**. GitHub может собирать стандартные серверные логи. Подробнее: [Политика конфиденциальности GitHub](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement).

---

### Удаление данных

- **Веб-браузер**: очистите данные сайта nstp3.github.io в настройках браузера
- **Android APK**: удалите приложение, или Настройки → Приложения → Razl → Хранилище → Очистить данные

---

Проект с открытым исходным кодом. Вопросы? Откройте issue на [GitHub](https://github.com/Nstp3/nstp3.github.io).
