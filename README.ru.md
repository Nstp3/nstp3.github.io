Личный дашборд продуктивности в стиле RPG. Превращает повседневные задачи, привычки и навыки в игровую прокачку персонажа. Работает прямо в браузере — без установки, без сервера, без регистрации.

---

## ✨ Возможности

- **Герой** — профиль с аватаром, уровнем, опытом и значком игрового кубка
- **Статы** — Здоровье, Настроение, Выносливость, Мотивация с прогресс-барами
- **Задачи** — добавление по категориям, повторяющиеся задачи, выполнение даёт XP
- **Навыки** — прокачка по 6 направлениям с визуализацией прогресса
- **Радар** — паутинка, отражающая текущий баланс навыков
- **Привычки** — трекер в формате календаря с drag-выделением и подписью дней
- **Помодоро** — точный таймер с настройкой рабочего / перерывного времени
- **Активность** — линейный график за последние 14 дней
- **3 темы** — Стандартная · Assassin's Creed · Solo Leveling
- **Экспорт / Импорт** — бэкап прогресса в JSON
- **Мультиязычность** — RU / EN
- **Уведомления** — помодоро + ежедневные напоминания о задачах и привычках (веб + Android)
- **Фильмы** — личный список с постером, фильтрами статусов и избранным
- **Игры** — библиотека с обложкой, ссылкой на источник, трекером прохождения и системой кубков

## ✅ Задачи — Повторяющиеся задачи

Любую задачу можно сделать **повторяющейся** — она будет автоматически создаваться каждый новый день.

### Как создать повторяющуюся задачу
Выбери **♻️ Повтор** в выпадающем списке категорий перед добавлением задачи.

### Как управлять повторяющимися задачами

| Действие | Как |
|---|---|
| Убрать флаг повтора | Нажми на иконку 🔄 у задачи |
| Переключить повтор (мобиле) | **Зажми задачу** на ~0.6 секунды → телефон завибрирует → флаг переключится |

> Повторяющиеся задачи появляются каждый день в списке. Выполнение начисляет XP как обычно.

## 🎮 Игры — Система кубков

Отслеживай пройденные игры. Кубок отображается в блоке «Игры» и **дублируется рядом с именем персонажа** в профиле.

| Пройдено | Кубок |
|---|---|
| 0–9 | — (шкала прогресса к 10) |
| 10+ | 🥉 Бронза |
| 50+ | 🥈 Серебро |
| 100+ | 🥇 Золото |
| 250+ | 💎 Платина |
| 500+ | 💜 Фиолетовый |
| 1000+ | 👑 Легендарный |

Каждый новый кубок **заменяет** предыдущий — и в блоке «Игры», и в карточке профиля.

## 📱 Android APK

Нативная Android-версия. Работает **офлайн** — все файлы встроены в приложение.

### Скачать

[**⬇️ Скачать app-release.apk**](android_version/app-release.apk)

### Установка

1. Скачай файл `app-release.apk`
2. Открой файл на телефоне
3. Если появится предупреждение → **Настройки → Безопасность → Разрешить установку из неизвестных источников**
4. Нажми **Установить**
5. Иконка **Nstp3 RPG** появится на рабочем столе

> ⚠️ APK собран вручную и не проходил проверку Google Play — для личного использования это нормально.

2. Нажми **«Поделиться»** → **«На экран "Домой"»**
3. Подтверди — иконка появится на рабочем столе

## 🔔 Уведомления

### Веб (браузер)
- При первом открытии браузер спрашивает разрешение
- Нажми **«🔔 Нажми чтобы включить уведомления»** под блоком Помодоро
- **Помодоро**: оповещение при окончании сессии или перерыва — работает на другой вкладке
- **Каждый день в 20:00**: напоминание о невыполненных задачах и привычках (пока вкладка открыта)

### Android APK
- При первом запуске появится системный диалог — нажми Разрешить
- **Помодоро**: нативный будильник через `AlarmManager` — срабатывает даже когда приложение закрыто
- **Каждый день в 20:00**: фоновая задача через `WorkManager` — срабатывает даже когда приложение закрыто

## ⏱ Помодоро

Таймер использует `Date.now()` внутри — отслеживает реальное время окончания, а не считает тики. Благодаря этому отображение остаётся точным даже когда браузер замедляет фоновые таймеры.

| Кнопка | Действие |
|---|---|
| Старт | Начать рабочую сессию |
| Пауза | Заморозить оставшееся время |
| Старт (после паузы) | Продолжить с того места где остановился |
| Сброс | Вернуться к полной длительности |
| Применить | Сохранить новые настройки и сбросить таймер |

## 💾 Данные и конфиденциальность

Все данные хранятся **локально на устройстве** в IndexedDB (~500 MB). Ничего не передаётся на серверы. Подробнее: [Политика конфиденциальности](PRIVACY.md).

- **Экспорт** — кнопка «Экспорт» → сохранит `life-rpg-backup.json`
- **Импорт** — кнопка «Импорт» → выбери сохранённый файл

> ⚠️ При очистке данных браузера прогресс сотрётся. Делай экспорт регулярно.

## 🚀 Локальный запуск

### Требования
- [Node.js](https://nodejs.org/) 18+

```bash
cd nstp3-rpg
npm install
npm run dev        # → http://localhost:5173
```

### Сборка для GitHub Pages
```bash
npm run build
```

### Быстрая пересборка APK
```bash
cd nstp3-rpg && \
BUILD_TARGET=android npm run build && \
rm -r ~/AndroidStudioProjects/Nstp3RPG/app/src/main/assets/* && \
cp -r dist-android/* ~/AndroidStudioProjects/Nstp3RPG/app/src/main/assets && \
echo "✓ Готово — собирай APK в Android Studio"
```

Затем: **Build → Generate Signed APK → release → Finish**
