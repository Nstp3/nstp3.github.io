Личный дашборд продуктивности в стиле RPG. Превращает повседневные задачи, привычки и навыки в игровую прокачку персонажа. Работает прямо в браузере — без установки, без сервера, без регистрации.

---

## ✨ Возможности

- **Герой** — профиль с аватаром, уровнем и опытом (XP / Daily XP)
- **Статы** — Здоровье, Настроение, Выносливость, Мотивация с прогресс-барами
- **Задачи** — добавление по категориям, выполнение даёт XP
- **Навыки** — прокачка по 6 направлениям с визуализацией прогресса
- **Радар** — паутинка, отражающая текущий баланс навыков
- **Привычки** — трекер по дням месяца с drag-выделением диапазона и подписью дней
- **Помодоро** — таймер с настройкой рабочего / перерывного времени
- **Активность** — график активности по дням
- **3 темы** — Стандартная · Assassin's Creed · Solo Leveling
- **Экспорт / Импорт** — бэкап прогресса в JSON
- **Мультиязычность** — RU / EN
- **Уведомления** — помодоро + ежедневные напоминания о задачах и привычках (веб + Android)

## 📱 Android APK

Нативная Android-версия. Работает **офлайн** — все файлы встроены в приложение.

### Скачать

[**⬇️ Скачать app-release.apk**](android_version/app-release.apk)

### Установка

1. Скачай файл `app-release.apk`
2. Открой файл на телефоне
3. Если появится предупреждение → **Настройки → Безопасность → Разрешить установку из неизвестных источников**
4. Нажми **Установить**
5. Иконка **Nstp3 RPG** появится на рабочем столе

> ⚠️ APK собран вручную и не проходил проверку Google Play — для личного использования это нормально.

## 📱 Добавить на рабочий стол (без APK)

Сайт уже в интернете — просто открой и добавь на рабочий стол.

### Android — Brave / Chrome

1. Открой [nstp3.github.io](https://nstp3.github.io/) в **Brave** или **Chrome**
2. Нажми **⋮** → **«Добавить на главный экран»** или **«Установить приложение»**

### iPhone — Safari

1. Открой [nstp3.github.io](https://nstp3.github.io/) в **Safari**
2. Нажми **«Поделиться»** → **«На экран "Домой"»**
3. Подтверди — иконка появится на рабочем столе

## 🔔 Уведомления

### Веб (браузер)
- При первом открытии браузер спросит разрешение на уведомления
- Нажми **«🔔 Нажми чтобы включить уведомления»** под блоком Помодоро
- **Помодоро**: оповещение когда сессия или перерыв закончились (работает на другой вкладке)
- **Каждый день в 20:00**: напоминание о невыполненных задачах и привычках (пока вкладка открыта)

### Android APK
- При первом запуске появится системный диалог разрешения
- **Помодоро**: нативный будильник через `AlarmManager` — срабатывает даже когда приложение закрыто
- **Каждый день в 20:00**: фоновая задача через `WorkManager` — срабатывает даже когда приложение закрыто

## 💾 Данные и конфиденциальность

Все данные хранятся **локально на устройстве** в IndexedDB (~500 MB). Ничего не передаётся на серверы. Подробнее: [Политика конфиденциальности](PRIVACY.md).

- **Экспорт** — кнопка «Экспорт» в топ-баре → сохранит `life-rpg-backup.json`
- **Импорт** — кнопка «Импорт» → выбери сохранённый файл

> ⚠️ При очистке данных браузера прогресс сотрётся. Делай экспорт регулярно.

## 🚀 Локальный запуск

### Требования
- [Node.js](https://nodejs.org/) 18+

```bash
cd nstp3-rpg
npm install
npm run dev        # → http://localhost:5173
```

### Сборка для GitHub Pages
```bash
npm run build      # результат: dist/
```

### Сборка для Android APK
```bash
BUILD_TARGET=android npm run build   # результат: dist-android/
```

### Быстрая пересборка APK (одна команда)
```bash
cd nstp3-rpg && \
BUILD_TARGET=android npm run build && \
rm -r ~/AndroidStudioProjects/Nstp3RPG/app/src/main/assets/* && \
cp -r dist-android/* ~/AndroidStudioProjects/Nstp3RPG/app/src/main/assets && \
echo "✓ Готово — собирай APK в Android Studio"
```

Затем: **Build → Generate Signed APK → release → Finish**
